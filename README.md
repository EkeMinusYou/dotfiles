## Pre-Setup

install

- [Homebrew](https://brew.sh/)

### Usage

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/EkeMinusYou/dotfiles/main/install.sh)"
```

#### Install Brew Packages

```bash
brew bundle --file "$HOME/src/github.com/EkeMinusYou/dotfiles/Brewfile"
```

### Contents

- Brewfile
- zsh setting
- Git config
- Neovim setting

- zsh - shell
